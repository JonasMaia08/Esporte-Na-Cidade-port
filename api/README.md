# Projeto Esporte na Cidade


este projeto é um sistema de gestão de alunos para uma iniciativa de aulas de esporte no município de potim
tendo as principais funcionalidades de:
- cadastro e gestao de alunos e professores
- registro de faltas
- realização de chamadas
   
Projeto de gestao de alunos e professores desenvolvido na faculdade FATEC - guaratingueta
